# To-Do List Manager with MongoDB

This project is a Python-based To-Do List Manager that uses MongoDB Atlas for cloud database storage. It demonstrates insert, update, delete, and retrieve operations, along with a related collections structure (`users` â†” `tasks`).

## Features
- Create users
- Add tasks tied to specific users
- View all tasks for a user
- Update task titles
- Delete tasks

## Setup

### Requirements
- Python 3.10+
- pymongo
- dnspython

### Install dependencies
pip install -r requirements.txt
## How to Run
python main.py

## Cloud Database
- MongoDB Atlas
- Database: `todo_database`
- Collections: `users`, `tasks`

## Note
Keep your MongoDB credentials secure and avoid uploading them to public repositories.
